Detailed Description

Base-Class for all the different devices
    - Xtender
    - VarioTrack
    - VarioString
    - RCC, Xcom-232i
    - Xcom-CAN
    - BSP
Inspired from and Based on hesso-valais/scom : scomdevice.py
<https://github.com/hesso-valais/scom/blob/0.7.3/src/sino/scom/device/scomdevice.py>
